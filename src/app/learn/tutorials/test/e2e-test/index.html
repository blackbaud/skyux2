Run an end-to-end test for a project

End-to-end tests identify system dependencies and ensure that an application flows as expected from start to finish. When you create a SKY UX project, the template includes a sample end-to-end test in the new project. You can follow the steps in this tutorial to run the end-to-end test for the default single-page application.

For information about end-to-end tests in SKY UX, see end-to-end tests. And for tips and techniques on testing Angular applications, see the Angular 2 testing guidelines.

  1. To view the end-to-end test spec file in a local editor, open the project's about.component.e2e-spec.ts file in the \e2e folder at the root of the project.

  2. In the command prompt, if skyux serve is still running, enter Ctrl-C to stop serving the project.

  3. Enter skyux e2e to run the end-to-end test.

  4. The test runs, and the command prompt displays the results.
