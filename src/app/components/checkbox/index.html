The checkbox component is a SKY UX-themed replacement for the HTML input element of type="checkbox". The value that is selected is driven through the ngModel attribute specified on the input element.

Accessibility

When using multiple checkboxes, use fieldset and legend elements to provide semantic structure. For people using screen readers, the fieldset element groups the checkboxes indicating the number of items in the group and the legend provides a description for the grouping. For more information on how to use fieldset and legend, see WebAIM examples and guidance.
