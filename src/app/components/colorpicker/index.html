The colorpicker module allows users to use a button to select a color. When this attribute is placed on an (input or button) element and wrapped in a sky-colorpicker component, the Element and colorpicker are created. This attribute must be set to the instance of the sky-colorpicker. This directive will use NgModel for binding data. Will return an object with the following interface: 
{{'{'}}
  hslaText: string
  rgbaText: string
  cmykText: string
  hsva: {{'{'}} hue:  number, saturation: number, value: number,      alpha: number{{'}'}}
  rgba: {{'{'}} red:  number, green: number,      blue: number,       alpha: number{{'}'}}
  hsla: {{'{'}} hue:  number, saturation: number, lightness: number,  alpha: number{{'}'}}
  cmyk: {{'{'}} cyan: number, magenta: number,    yellow: number,     key: number  {{'}'}}
  hex:  string
  {{'}'}}
 Specifies a array of colors that will load in the colorpicker. Specifies the initial color to be used, default #FFF. Specifies the value property on the provided input or button, accepts rgba, hex or hsla
