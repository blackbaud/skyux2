As a general rule, SKY UX uses color to convey meaning and not as pure decoration. You should rarely apply colors directly to elements because components incorporate colors as necessary. If a need for color arises outside of a component, you likely need to create a component for that scenario. Text For text, ensure that the contrast between the text color and the background color is at least 4.5:1 to meet Web Content Accessibility Guidelines 2.0 level AA.

Modified states

In general, hover states for active elements apply a 10 percent darken to the element's background, border, and/or text.

In general, disabled states for active elements apply .65 opacity to the element.
