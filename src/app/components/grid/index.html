This grid component displays a SKY UX-themed grid of data. The grid should contain sky-grid-column components that specify column information. The sky-grid component can provide a view of a searchable, paginated list component when used within the sky-list component. Specifies an array of data for the grid. Each item in the array should have an ID and a property that maps to either the field or id property of each column in the grid. Specifies an array of columns to display in the grid. Will match the id or field properties of the columns. If no array is provided, then the grid displays all columns. Acceptable values: Array of strings. Specifies how the grid should fit to its parent. Acceptable values: 'width' or 'scroll'. Specifies the height of the grid. Acceptable values: number. Specifies the width of the grid. Acceptable values: number. Specifies the field to use to sort the grid. Acceptable values: ListSortFieldSelectorModel. ListSortFieldSelectorModel contains the fieldSelector string that represents the current sort field and the descending boolean that indicates whether to sort in descending order. Specifies a template for the row details. Adds a dropdown at the end of each table that when clicked, shows the details about the row. The template has access to the row variable, which contains the entire row data. Fires when the columns to display in the grid change. Emits an array of column IDs. Fires when the active sort field changes. Emits a ListSortFieldSelectorModel. Specifies a unique ID for the column. If no field property exists on the column, then this is used as the property to retreive cell information from an entry on the grid data array. Specifies the property to retreive cell information from an entry on the grid data array. If no id property exists on the column, then this will be used as the entry for the grid selectedColumnIds array. Specifies text to display in the column header. Specifies a description for the column. Specifies the width of the column. Indicates whether the column is locked. If set to true, then users cannot drag the column to another position and or drag other columns before the locked column. Indicates whether the column sorts the grid when users click the column header. Indicates whether the column is initially hidden when grid selectedColumnIds are not provided. Specifies a search function to apply for the specific column. By default, the column executes a string compare on the column data. Acceptable values: (data: any, searchText: string) => boolean Specifies a template for a column. This can be assigned as a reference to the template attribute, or it can be assigned as a child of the template element inside the sky-grid-column component. The template has access to the value variable, which contains the value passed to the column, and the row variable, which contains the entire row data.
