Edit a component on an existing page

After you edit an existing page in the default single-page application, you can follow these steps to update an existing component on that page.

  1. With the skyux serve command running, return to the default site's About page in your web browser. The paragraph from the previous tutorial should be in place.

  2. In a local editor, return to the project’s about.component.html file.

  3. Edit the type and message for the alert component. For example, in the alertType attribute, change the value to from warning to info. And in the sky-alert element, change the message to <strong>Note:</strong> There may be a delay in responding to your email.

  4. Save the changes.

  5. Return to the command prompt to see that the skyux serve command recognizes the changes and updates the page.

  6. Return to the About page in the web browser to see the update in place. The new alertType value updates the background color, and the new text replaces the previous message.
