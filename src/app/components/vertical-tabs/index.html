The vertical tabs module contains components to render a vertical tabset.

The text of the button that selects the tab. Displays a counter alongside the tab header. Determines whether the tab is disabled. Specifies the text to use for the show tabs button in mobile. Specifies the text to use in the heading of the collapsible group of tabs. Boolean value that can be set to disabled expanding and collapsing a collapsible group. Boolean value indicating if the collapsible group is open. Fires when the active tab changes. Emits the index of the active tab.
