The SKY UX tutorials get you up and running with SKY UX. They demonstrate how to create and serve single-page applications, how to create and edit pages and components, how to run tests, and how to host projects.

These tutorials currently only cover the basics. We will create additional tutorials as we identify areas where they are necessary. Prerequisites

The first step to set up your machine for SKY UX is to make sure you have the following prerequisites in place:

  • Node.js version 6+ — To check your version, run node –v from the command line.

    Do not use Node.js version 7+.

  • NPM version 3+ — NPM is the default JavaScript package manager for Node.js. As of Node.js version 0.6.3, it is bundled and installed automatically with the environment. We recommend version 3+ for its flat dependency structure. To check your version, run npm –v from the command line.

  • Git — To confirm that you have Git in place, run git version from the command line.
