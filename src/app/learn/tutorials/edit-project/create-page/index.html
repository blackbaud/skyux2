Create a page

This tutorial walks through the process to create a page in a single-page application. By default, SKY UX uses automatic route configuration based on the project's folder structure. This means that to create a page, you just create a folder in the existing \src\app folder and add an index.html file to that new folder.

  1. Return to the project in a local editor, and right-click the app folder.

  2. Click New Folder, and then rename the folder. The folder name appears in the route for the page. For the tutorial, use "demo" for the folder name.

  3. Right-click the new demo folder, and click New File.

  4. Rename the new file "index.html."

  5. In the new file, place the shared app-nav component at the top and insert hello world content such as <h1>Hello World!</h1>. The app-nav component is an example of a shared component. It manually creates top-level navigation within your app.

  6. Save the changes to the new index.html file.

  7. Go to the shared folder, and open the app-nav.component.ts file.

  8. Add a path for the new page with name set to "Demo" and path set to "/demo." The name is the label for the menu, and the path is the name of the folder.

  9. Save the changes to the component file.

  10. If skyux serve is running, it recognizes the changes and updates the SPA. If not, open the command prompt and enter skyux serve to launch the SPA.

  11. Return to the default site to see the update in place. The Demo item appears in the menu.

  12. Click the Demo menu option. The new page appears.
