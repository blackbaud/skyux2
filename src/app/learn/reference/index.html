The reference section provides technical resources to help manage SKY UX projects, including SKY UX CLI commands, configuration options, and file placement and naming conventions.

Keep in mind that Angular is at the heart of SKY UX 2, so you need to be familiar with this technology to build with SKY UX. For learning resources for Angular and other SKY UX tools and technologies, see the training resources.

Keep in mind that some options in the references are for internal Blackbaud use only. External developers can use SKY UX to create, test, and run SKY UX SPAs. And SKY UX components, services, and modules are open source and available in the SKY UX 2 GitHub repo. However, not all SKY UX features are available for external use.
