The tabs module contains components to render a tabset and includes options to display buttons in the tab area and to make tabs closeable.

The index of the active tab. Fires when the active tab changes. Emits the index of the active tab. Fires when the user clicks the "new tab" button. Fires when the user clicks the "open tab" button. Specifies the tab header. Displays a counter alongside the tab header. The index of the tab. If not defined, it will be set to the position of the tab on load. Indicates whether to disable the tab. Specifying a listener for this event adds a close button to the tab. The event fires when the user clicks the tab's close button.
