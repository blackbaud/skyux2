Technical reference

The technical reference includes resources to describe the commands and options that are available in the SKY UX CLI and configuration options for the SKY UX process.

  • For commands and options in the SKY UX CLI, see the CLI commands reference.

  • For settings that are available in the skyuxconfig.json file in your project's directory, see the configuration options reference.

  • For helpers to make authenticated HTTP requests, see the helpers reference.
