The timepicker module contains directives for selecting times. To use as a SKY UX-themed replacement for the HTML input element with type="time". The value that is selected is driven through the ngModel attribute specified on the input element. Specifies 12-hour hh or 24-hour HH format. Specifies a custom time format. See the momentjs docs for examples. When this attribute is placed on an input element and wrapped in a sky-timepicker component, the input and time dropdown are created. This attribute must be set to the instance of the sky-timepicker. This directive will use NgModel for binding data.
