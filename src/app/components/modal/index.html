The modal directive and service allow you to launch modals in a consistent way in SKY UX applications. The SkyModalService provider lauches the modals, while the sky-modal directive provides a common look-and-feel for modal content. Within the sky-modal directive, you have options to display a common modal header, specify the body content, and display a common modal footer and buttons. The SkyModalInstance class allows you to close the modal and return data from the launched modal.
