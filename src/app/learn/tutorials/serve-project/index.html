Serve a project

This tutorial walks through the process to serve a SKY UX single-page application.

  1. From the command prompt, cd into the root directory for the SKY UX project we created in the previous tutorial.

  2. Enter skyux serve to launch the SPA. It opens in your default browser.

  3. In the command prompt, the command continues to run, and changes to your project are automatically displayed as you save them.
