Edit an existing page

After you serve the default single-page application, you can follow these steps to update an existing page.

  1. With the skyux serve command running, open the default site's About page in your web browser.

  2. In a local editor, open the project's about.component.html file in the \src\app\about folder.

  3. At the top of the file before the warning, add a paragraph such as <p>This page provides contact information for members of our team.</p>.

  4. Save the changes.

  5. Return to the command prompt to see that the skyux serve command recognizes the changes and updates the page.

  6. Return to the About page in the web browser to see the new paragraph in place.
