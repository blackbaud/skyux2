The autonumeric pipe formats numbers to a shortened value if it is 1000 or more to a rounded number plus an abbreviation character. K for thousands, M for millions, B for billions and T for trillion. It will also format the shortened number as a currency as an option.

Pipe Usage: number_expression | skyAutonumeric[:digits[:currency[:iso]]]

It can also be imported and the transform method called in ts files.
Method Usage: this.skyAutonumeric.transform(value, digits, currency, iso4217) Number. Specifies a value to be shortened. Number. Specifies the maximum number of fractional digits to round to and display. Boolean. Specifies if the value should be formatted as Currency as well as shortened. String. Specifies the ISO4217 currency code to be used for currency formatting.
