The form classes create styled inputs and labels for use in forms.

Form control

The sky-form-control class consistently styles the input and allows it to fill the width of its container.

Control label

The sky-control-label class consistently styles labels for form input.

Required fields

The sky-control-label-required class indicates that a given input is required to submit a form.

Error states

Form fields that use Angular validation will by styled with a red glow when they are in an invalid state. We specifically check for the following conditions to apply the styling:

  • input fields with classes ng-invalid + ng-touched
  • select fields with classes ng-invalid + submitted

The sky-error-label class provides styling for error messages for a form field that is in an invalid state. You should place the message below the form field, and include logic to show the message when the conditions for showing invalid styling are met.

Form group

The sky-form-group class provides consistent vertical spacing between fields.
