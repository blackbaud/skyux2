The text expand component truncates long text with an ellipsis and a link that allows the user to fully expand the text. If the text length falls below the specified threshold, then no action is taken. The 'See more' link will expand the text inline if it does not exceed the limits on text characters or newline characters for the inline expanded view. Otherwise, the link will open the text in a modal view.

Note that collapsed text will have newlines removed. Also, if newlines are detected, the text is automatically collapsed regardless of the total length.

The text expand repeater component truncates a list of repeater items and will initially display a set number of items. Any items over the set maximum limit are hidden until the user expands the list.

The text to truncate. The number of characters to show before truncating the text. To avoid truncating text in the middle of a word, the directive looks for a space in the 10 characters before the last character. The maximum number of characters to show in the inline expanded view. If the text includes more characters, then a modal view with the content will be displayed when the 'See more' link is clicked The maximum number of newline characters to show in the inline expanded view. If the text includes more newline characters, then a modal view with the content will be displayed when the 'See more' link is clicked. The title to display in the modal expanded view. The data to truncate. The number of items from the data to display before truncating.
