The datepicker module allows users to use an input and calendar to select dates. When this attribute is placed on an input element and wrapped in a sky-datepicker component, the input and calendar are created. This attribute must be set to the instance of the sky-datepicker. Specifies a maximum selectable date in the calendar. Specifying this as an attribute on the input will override the default in SkyDatepickerConfigService.
