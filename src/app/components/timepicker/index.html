The timepicker module contains directives to let users select times. It is a SKY UX-themed replacement for the HTML input element with type="time". The value that users select is driven through the ngModel attribute specified on the input element. Specifies 12-hour hh or 24-hour HH format. Specifies a custom time format. See the momentjs docs for examples. When this attribute is placed on an input element and wrapped in a sky-timepicker component, the input and time dropdown are created. This attribute must be set to the instance of the sky-timepicker. This directive uses NgModel to bind data.
