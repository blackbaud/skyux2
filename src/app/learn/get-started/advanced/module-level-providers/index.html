You can import module-level providers and services to supplement SKY UX functionality and meet needs that SKY UX does not handle. You import providers and services in the app-extras.module.ts file, similar to how you add entry components.

To illustrate how to import module-level providers and services, the code sample below imports DragulaService and DragulaModule into the app-extras.module.ts file. The DragulaService and DragulaModule in this example provide a set of components and services to make drag and drop easier.

In the code sample, we add DragulaModule to both the imports and exports properties of AppExtrasModule because app-extras.module.ts is not the application's root module. import { NgModule } from '@angular/core'; import { DragulaService, DragulaModule } from 'ng2-dragula/ng2-dragula'; // Specify entry components, module-level providers, etc. here. @NgModule({ imports: [ DragulaModule ], exports: [ DragulaModule ], providers: [ DragulaService ], entryComponents: [] }) export class AppExtrasModule { }
