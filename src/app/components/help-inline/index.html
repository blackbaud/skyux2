The help inline component creates a small button next to a field. When clicked they can trigger an action that displays more information about that field that the user might need information on.

Fires when the user clicks the help inline button

Use the help inline component to provide information about a field or section such as how a value is calculated. Use this option when persistent inline help is not necessary.

You can open the content in a popover or the help panel, depending on how extensive the information is.
