A wizard modifies a modal component with tabs to guide users through a set of pre-defined steps. A wizard is useful when users need to perform steps in a particular order. To display a series of tabs as a wizard, set tabStyle=”wizard” on the sky-tabset component. For more information about tabs, see the tabs module. Specifies the behavior for a series of tabs.
