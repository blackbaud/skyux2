For a group of radio buttons, use fieldset and legend elements to provide semantic structure. For people using assistive technology, the fieldset element groups the radio buttons to indicate the number of items in the group, and the legend element provides a description for the group. For more information on how to use fieldset and legend, see WebAIM examples and guidance.
