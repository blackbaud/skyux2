--assets — Specifies where to locate static assets when skyux build runs. The skyux build command replaces any instances of ~/assets/ with this value in HTML, CSS, and SCSS files. For example, skyux build –-assets https://example.com/assets. By default, the command replaces ~/assets/ with the CDN. For information about how to include static assets in HTML, CSS, and SCSS files, see the static assets tutorial.
