The popover component allows you to display contextual content when a user clicks a button or link. The popover's content can be plain text, HTML, or any Angular component. A popover can be closed by clicking on the the trigger element that called it, clicking anywhere on the page, or by using the skyPopoverClosed directive described below. References a SkyPopoverComponent instance on the page. Add this directive to an element that will open a popover when clicked. Specifies the placement of the popover in relation to the trigger element (above, below, left, right). Specifies the user action that displays the popover (click, mouseenter). Adds a title to the popover. The template reference variable used by the [skyPopover] directive. Fires when the user opens the popover. Fires when the user closes the popover.

To ensure usability on touch devices, you should generally trigger popovers on click rather than on hover.
