Creates an input control for users to enter search criteria. Specifies search text for the input. Specifies placeholder text for the search input. Specifies the expand mode for the search input. The valid options are responsive, none, and fit.
  • responsive The search input collapses into a button on mobile devices.
  • none The search input does not collapse into a button on mobile devices.
  • fit The search input fits the width of its container.
 Fires when the search text is applied. Fires when the search text is changed. Fires when the search text is cleared.
