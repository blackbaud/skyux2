Default setup

These tokens can be navigated with the arrow keys, and they can be dismissed with the close buttons.

Non-dismissible tokens

These tokens cannot be dismissed, but they can still be navigated with the arrow keys.

Non-focusable tokens

These tokens cannot be focused using the tab key, but they can still be clicked and navigated with the arrow keys.

Disabled tokens

These tokens are disabled and cannot be dismissed or navigated with the arrow keys.
