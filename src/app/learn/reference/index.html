The reference section provides technical resources to help manage SKY UX projects. The resources include descriptions of the SKY UX CLI commands and options, configuration options for the SKY UX process, and conventions for file placement files and naming.

Keep in mind that Angular is at the heart of SKY UX 2, so you need to be familiar with this technology to build with SKY UX. For learning resources for Angular and other SKY UX tools and technologies, see the training resources.
