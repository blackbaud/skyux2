The error component displays a SKY UX-themed error message. It includes options to specify the image, title, description, and action. Specifies an image to display in the error message. Specifies a title to display in the error message. Specifies a description to provide additional details in the error message. Specifies an action to include in the error message. For example, you can include a button to reload the page or to refresh data. An attribute to specify a set of pre-defined values for the image, title, and description. The valid options are broken, not-found, and construction. This event will be emitted when the action button is clicked.
