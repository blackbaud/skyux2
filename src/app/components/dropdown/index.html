The dropdown component renders a button that when clicked will display a dropdown menu.

Specifies the type of button to render as the dropdown's trigger element. Specify select to display a button with text and a caret or context-menu to display a round button containing an ellipsis. If you use the select style you will need to include a sky-dropdown-button element to specify the text for the button. Specifies the accessibility label text for the dropdown button when the dropdown button has no text. Specifies what user action will cause the dropdown to open. Possible values are click or hover. Specifies whether the card is selected. This property only applies when the card's selectable property is true.
