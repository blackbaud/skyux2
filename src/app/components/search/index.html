Creates an input control for users to enter search criteria. On mobile devices, the component displays a search button that users can click to display the input control. Specifies search text for the input. Specifies placeholder text for the search input. Specifies whether the search input should collapse into a search button on small screens. Specifies whether the search input should take up the width of its entire container. Note that setting this to true will prevent the collapsing functionality. Specifies whether the search background has a dark theme when search is full width or displayed in collapsed mode. Fires when the search text is applied. Fires when the search text is changed.
