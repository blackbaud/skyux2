Add a component to an existing page

After you edit a component on an existing page in the default single-page application, you can follow these steps to add a SKY UX component on that page.

  1. With the skyux serve command running, return to the default site's About page in your web browser. The updates to the alert component from the previous tutorial should be in place.

  2. In a local editor, return to the project’s about.component.html file.

  3. Between the alert and avatar components, add a card component. Add the sky-card element, and then within that element, add a sky-card-title element for a title, a sky-card-content element for the content, and a sky-card-actions element for a button.

    <sky-card>
  <sky-card-title>Large card</sky-card-title>
  <sky-card-content>
    This card demonstrates the amount of space that is available for a card that uses
    the default large size. If the content does not require this much space, you can
    set the card size to small. The type of content to display in the body of a card
    varies based on what the card represents and whether it prompts users to action.
    The content can consist of some combination of a graph, trend, status, summary
    information, and guidance text.
  </sky-card-content>
  <sky-card-actions>
    <button class="sky-btn sky-btn-default">Click me</button>
  </sky-card-actions>
</sky-card>

  4. Save the changes.

  5. Return to the command prompt to see that the skyux serve command recognizes the changes and updates the page.

  6. Return to the About page in the web browser to see the new component in place. The card component appears between the alert and avatar components.
