This list component displays a SKY UX-themed list of data. It includes options for adding searching, paging.

Related Components

Here is a list of related components that can be used within the list component.

  • List Paging (sky-list-paging)
  • List Toolbar (sky-list-toolbar)
  • List View Checklist (sky-list-view-checklist)
  • List View Grid (sky-list-view-grid)
 Specifies the data to display. Must provide data or dataProvider. Acceptable values: Observable Array or Array (to update data in your view outside of a dataProvider you must use an Observable instead of a static array). Specific a data provider used to obtain data to display. Must provide data or dataProvider. Acceptable values: extends ListDataProvider. To extend ListDataProvider, provide a get function that accepts a ListDataRequestModel as an argument and returns a Observable ListDataResponseMode , and a count function that returns a Observable number that indicates how many total records are in the list. Specifies the total number of items for the initial data set when initialized. When used in conjunction with data and dataProvider it allows an initial data to be set with the need to call the dataProvider. Specific a function to apply as a global sort on the list. Acceptable value: (searchText: string) => Observable Array or Promise Array. Specific a set of selected ids to used with a checklist view. Acceptable values: Observable Array of strings or Array of strings. When using a list view that supports item selection, this list will emit this event with a Map string, boolean object indicates which data entries are selected.
