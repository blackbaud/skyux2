The form classes create styled inputs and labels for use in forms.

Form control

The sky-form-control class consistently styles the input and allows it to fill the width of its container.

Control label

The sky-control-label class consistently styles labels for form input.

Required fields

The sky-control-label-required class indicates that a given input is required to submit a form.

Error states

Form fields that use Angular validation are styled with a red glow when they are in an invalid state.

  • input fields with ng-invalid and ng-touched classes
  • select fields with ng-invalid and submitted classes

The sky-error-label class provides styling for error messages for form fields in an invalid state. Place the message below the form field, and include logic to display the message when the conditions for invalid styling are met.

Please enter a valid value.

Form group

The sky-form-group class provides consistent vertical spacing between fields.
