During local development with SKY UX, local files are loaded into SKY UX Host. To load your local single-page application into SKY UX Host, you must install the skyux-ca.crt root certificate. Follow the installation instructions appropriate for your machine.

Host is primarily for internal Blackbaud use. When external developers use the SKY UX CLI to serve SKY UX SPAs locally, Host loads local files. However, Host is not open source like the components, services, and modules in the SKY UX 2 GitHub repo. Installation for Macs

  1. Download the skyux-ca.crt.

  2. Open the Keychain Access application.

  3. Under the Keychains list on the left, select "login."

  4. Under the Category list, select "Certificates."

  5. Drag skyux-ca.crt into the list of certificates.

  6. Double-click the new "SKY UX" item in the list to open the certificate's info window.

  7. Expand the Trust section near the top of the info window, and select the "Always Trust" option for "Secure Sockets Layer (SSL)."

  8. Close the info window.

Installation for Windows

  1. Right-click the skyux-ca.crt URL, and select Save As.

  2. Open the file.

  3. On the security warning prompt, click Open.

  4. On the Certificate screen, click Install Certificate.

  5. On the Certificate Import Wizard, click Next.

  6. Select Place all certificates in the following store, and then click Browse.

  7. On the Select Certificate Store screen, select Trusted Root Certification Authorities, and then click OK.

  8. On the Certificate Import Wizard, click Next.

  9. Click Finish.

  10. On the security warning prompt, click Yes.

  11. On the Certificate Import Wizard confirmation prompt, click OK.

  12. After you return to the Certificate screen, click OK.
