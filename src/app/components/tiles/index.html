The sky-tile directive creates a collapsible container and is the bulding block for pages and forms in a Sky UX application. The sky-tile-section directive is used to create padded sections inside a sky-tile element. Additionally, the sky-tile-header-content directive may be placed inside the sky-tile directive to add summary information to the tile. If you simply need to show a check mark indicating the tile has data, you can add a sky-tile-header-check element to the sky-tile-header-content element.

When used on forms, it automatically adjusts the background color on the form and shrinks the tile header.
