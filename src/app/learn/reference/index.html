The technical reference includes resources to describe the commands and options that are available in the SKY UX CLI and configuration options for the SKY UX process.

Keep in mind that Angular 2 is at the heart of SKY UX 2, so you need to be familiar with this technology to build with SKY UX. For learning resources for Angular2 and other SKY UX tools and technologies, see the training resources.
  • For commands and options in the SKY UX CLI, see the CLI commands reference.
  • For a description of the default items that the template provides in a SKY UX single-page application, see the template items reference.
  • For settings that are available in the skyuxconfig.json file in your project's directory, see the configuration options reference.
  • For helpers to make authenticated HTTP requests, see the helpers reference.
  • For information about unit tests and end-to-end tests in SKY UX projects, see the testing reference.
