This list view grid component displays a SKY UX-themed grid of data. It includes the ability to defined a custom search function for the data given. Ideally it is used in conjunction with the list component. Specifies the name of the view. Acceptable values: string. Specifies the columns to be hidden by default. Will match off of the id or field of the item. Acceptable values: Array of strings or Observable Array of strings. Specifies columns to be displayed by default. Will match off of the id or field of the item. Acceptable values: Array of strings or Observable Array of strings. Specifies how the gird should fit to its parent. Acceptable values: 'width' or 'scroll'. Specifies the height of the grid. Acceptable values: number or Observable number. Specifies the width of the grid. Acceptable values: number or Observable number. Specify a search function to apply on the view data. Acceptable value: (data: any, searchText: string) => boolean. Specify if the grid will have a static selector column with checkboxes. Acceptable value: boolean or Observable boolean.
