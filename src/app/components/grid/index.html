This grid component displays a SKY UX-themed grid of data. Specifies columns to be displayed by the grid. Will match off of the id or field of the column. If no array is provided, then all columns are shown. Acceptable values: Array of strings. Specifies how the grid should fit to its parent. Acceptable values: 'width' or 'scroll'. Specifies the height of the grid. Acceptable values: number. Specifies the width of the grid. Acceptable values: number.
