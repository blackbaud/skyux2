The vertical tabs module contains components to render a vertical tabset.

Specifies the text to display on the show tabs button on mobile devices. Indicates whether the tab is active when the tabset loads. Specifies the tab header. Displays a counter alongside the tab header. Indicates whether to disable the tab. Specifies the header for the collapsible group of tabs. Indicates whether to disable the ability to open and collapse the collapsible group. Indicates whether the collapsible group is open. Fires when the active tab changes. Emits the index of the active tab. The index is based on the tab's position when it loads.
When to use vertical tabs
  • Use vertical tabs when a page has large amounts of content to divide into separate pages and standard tabs would extend past the edge of the screen.
  • Use vertical tabs when it is important to group related tabs into collapsible sections.
When not to use vertical tabs
  • The standard tab control is the preferred option to divide large subsets of content on a page. Do not use vertical tabs when the standard tab control will work.
  • Do not use more than one set of vertical tabs on a page.
  • Do not use vertical tabs on a form. Use the sectioned form component instead.
