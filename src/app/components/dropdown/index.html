The dropdown component renders a button that displays a menu when users click it. The menu displays a set of actions or functions. It is not a select input for users to set the menu's value to one of the options. If you need a selection input, see the options to select items from lists.

Specifies the type of button to render as the dropdown's trigger element. Specify select to display a button with text and a caret or context-menu to display a round button containing an ellipsis. You can also specify a font-awesome icon as the content of the dropdown by passing the class name to the buttonType attribute. For example, setting buttonType to filter would set the fa-filter icon as the content of the dropdown button. If you use the select style you will need to include a sky-dropdown-button element to specify the text for the button. Specifies the background color of the dropdown's trigger element. if unspecified it is set to default for background color sky-color-white. Specify primary to change color to sky-color-blue. Specifies the accessibility label text for the dropdown button when the dropdown button has no text. Specifies a title for the dropdown button when no text exists. Specifies what user action will cause the dropdown to open. Possible values are "click" or "hover". Provide an observable to send commands to the dropdown. The commands should respect the SkyDropdownMessage type (see below). Set this property to false to disable the browser's native focus function when navigating the menu items using a keyboard. This is useful when you wish to interact with the dropdown menu, but keep the keyboard focus on a different element. An observable that emits changes when the menu's active index changes. The response is of type SkyDropdownMenuChanges (see below).

Set the message to one of SkyDropdownMessageType:

  • SkyDropdownMessageType.Open — Opens the dropdown menu.
  • SkyDropdownMessageType.Close — Closes the dropdown menu.
  • SkyDropdownMessageType.FocusTriggerButton — Focuses the dropdown's trigger button.
  • SkyDropdownMessageType.FocusNextItem — Focuses the next item in the dropdown menu.
  • SkyDropdownMessageType.FocusPreviousItem — Focuses the previous item in the dropdown menu.
The currently active menu index.
