This list component displays a SKY UX-themed list of data. It includes options for adding searching, filtering, paging, and multiple view types.

Related Components

Here is a list of related components that can be used within the list component.

  • List Filters (sky-list-filters)
  • List Paging (sky-list-paging)
  • List Toolbar (sky-list-toolbar)
  • List View Checklist (sky-list-view-checklist)
  • List View Grid (sky-list-view-grid)
  • List View Repeater (sky-list-view-repeater)
  • List Action Bar (sky-list-action-bar)
