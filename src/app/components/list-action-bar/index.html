The list action bar component displays a customizable bar with hooks to the list component that give it access to the number of selected items, as well as the ability to select or clear all item selections. Specify whether the action bar will always be on, or only on selection. Acceptable values: boolean or Observable boolean. Getter function that returns the number of items currently selected. Return Type: number. Function that selects all items on the grid. This includes items not shown on the page in a paged grid. Return Type: void. Function that deselects all items on the grid. This includes items not shown on the page in a paged grid. Return Type: void. This list action bar item component defines a given action bar item. It is used in conjunction with the list action bar component. Specifies the template to be used to render the action bar item. Note: You must either define a template property, or define an inline template.
