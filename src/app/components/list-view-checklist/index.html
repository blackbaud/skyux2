This list view checklist component displays a SKY UX-themed checklist of data. It includes the ability to defined a custom search function for the data given. Ideally it is used in conjunction with the list component. Specifies the name of the view. Acceptable values: string. Specifies the name of the lebel field selector. Acceptable values: string. Specifies the name of the description field selector. Acceptable values: string. Specifies the name of the category field selector. Acceptable values: string. Specific a search function to apply on the view data. Default searches view data on label, description, and category if field selectors are defined for the given field. Acceptable value: (data: any, searchText: string) => boolean.
