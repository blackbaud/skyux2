The list filter module provides components around filtering lists. This component should contain a filter button for the list toolbar. Use the sky-filter-button component internally to open a modal with filtering options. Apply filter options by setting the appliedFilters input on sky-list. Including this component inside of sky-list-toolbar creates a filter summary based on the list's appliedFilters input. This component creates an inline filter area for the list.
