The media queries service allows users to subscribe to screen size changes at different breakpoints. Suscribes to screen size changes. The listener function is called when breakpoints change. It is called with a SkyMediaBreakpoints argument, which is an enum that represents the new breakpoint. The possible values for the argument are xs for screen widths of 767px or less, sm for screen widths of 768px to 991px, md for screen widths of 992px to 1199px, and lg for screen widths of 1200px or greater).
