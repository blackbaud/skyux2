SKY UX 2 is an updated version of Blackbaud's UX framework. The framework brings a consistent, cohesive experience to Blackbaud products and allows third-party developers to provide the same consistent experience in their customizations and applications. SKY UX 2 takes advantage of updates to Angular to keep SKY UX modern while increasing the tooling, testing, and performance available.

To follow along with the development of SKY UX and to make contributions, see the SKY UX 2 GitHub repo. The SKY UX 2 documentation lives alongside the components in the repo, so you can contribute to the documentation there as well. For information about how to contribute, see the SKY UX 2 contribution guidelines.
