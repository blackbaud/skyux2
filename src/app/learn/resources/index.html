Training resources

Angular 2 is at the heart of SKY UX 2, so you need to be familiar with this technology to build with SKY UX. In addition, you should be familiar with several other tools and technologies. This page provides recommended learning resources for some of these tools.

Key tools and technologies

  • Angular 2 — Angular 2 is the key building block of SKY UX 2. Check out the Angular docs, and see Learning Angular for a suggested path for Angular newcomers. A curated list of Angular 2 resources provides additional tutorials, courses, and other learning material. And for a guide to Angular syntax, conventions, and application structure, see the Angular 2 Style Guide.

  • Node.js — Node is a JavaScript runtime that is a prerequisite for SKY UX Builder. Node.js uses an event-driven, nonblocking I/O model to build scalable network applications. For more information, see the Node.js website.

  • NPM — NPM is a package manager for JavaScript that runs through the command line and manages dependencies. NPM is the default for Node.js, and it is a prerequisite for SKY UX Builder. For more information, see the NPM website.

  • Git — Git is an open source distributed version control system, and you need to create Git repos for your SKY UX 2 projects. For more information, see the Git documentation, explore the introduction tutorial, or check out GitHub's hello world project.

  • TypeScript — TypeScript is a typed superset of JavaScript that compiles to plain JavaScript. For more information, see the TypeScript website and the quick start tutorial.

  • Sass — Sass is a CSS extension language that has two syntaxes. SASS is the older, indented syntax that uses the .sass extension. SCSS is the more common syntax that is a superset of CSS3 syntax, and it uses the .scss extension. For more information, see the SASS website.

Additional resources

  • Fat-arrow functions — Arrow functions are a syntax for JavaScript functions that save time and simplify function scope.

  • Observables — Angular observables are similar to promises, but observables are cancellable and can handle multiple values over time. For more information, see this observables example and this observables video.

  • Flexbox CSS guide — Flexbox, or CSS Flexible Box, is a layout mode that arranges elements on a page so that they behave predictably for different devices and screen sizes.
