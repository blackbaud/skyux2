A wizard uses tabs to guide users through a set of pre-defined steps. A wizard is useful when users need to perform steps in a particular order. To display a series of tabs as a wizard, set tabStyle="wizard" on the sky-tabset component. To include buttons for users to navigate through the wizard, use the sky-tabset-nav-button component. For more information about tabs, see the tabs module. Specifies the behavior for a series of tabs. To display tabs as steps in a wizard, set tabStyle="wizard" on the sky-tabset component. Specifies the button type for a tabset nav button. To create a Previous button, set buttonType="previous" on the sky-tabset-nav-button component. To create a Next button, set buttonType="next" on the sky-tabset-nav-button component. Specifies the sky-tabset component that the tabset nav button controls.
