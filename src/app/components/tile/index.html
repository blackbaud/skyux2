The sky-tile directive creates a collapsible container and is the bulding block for pages and forms in a SKY UX application. The sky-tile-section directive is used to create padded sections inside a sky-tile element. Additionally, the sky-tile-summary directive may be placed inside the sky-tile directive to add summary information to the tile. Any sky-tile components used within a sky-tile-dashboard must be included as entry components in the application module.

Indicates whether the tile is in a collapsed state. Event emitted when the settings button on the tile is clicked. The settings button will only appear if this output is placed on the tile. Event emitted when the tile changes collapsed state. Will return true if collapsed, false otherwise. The SkyTileDashboardConfig object that populates the tile dashboard. This event emits a SkyTileDashboardConfig when the tile dashboard changes state. This can occur when tiles are collapsed/expanded or when tiles are rearranged via drag and drop. An array of SkyTileDashboardConfigTile objects that indicates which tiles are contained in the dashboard. Each SkyTileDashboardConfigTile has the following properties:
  • id The unique identifier for the tile.
  • componentType The class type for the tile component.
  • providers An array of data providers that can be passed to the tile.
 A SkyTileDashboardConfigLayout object that describes the layout of the tile dashboard. The SkyTileDashboardConfigLayout object contains a singleColumn property of type SkyTileDashboardConfigLayoutColumn and a multiColumn property which is an array of type SkyTileDashboardConfigLayoutColumn. The singleColumn layout will display on small screens, while the multiColumn layout will display on larger screens. The SkyTileDashboardConfigLayoutColumn object contains an array of SkyTileDashboardConfigLayoutTile objects which indicate the order of tiles and have the following properties:
  • id The identifier for the tile in the dashboard.
  • isCollapsed A boolean value indicating whether the specific tile is in an expanded or collapsed state.
