The filter module provides components that allow users to select filter criteria. These components create a button to expose filtering options, a summary to indicate which filters have been applied, and a area for filtering inline. Indicates whether the filter button should be given active indication. This property should be set when indication of filtering is not visible to the user, such as when a filter summary is not shown. An event that is emitted when the filter button is clicked. Indicates whether the filter summary item has a close button. An event that is emitted when the summary item close button is clicked. An event that is emitted when the summary item is clicked.
